# npm 代理加载

```
npm --registry=https://registry.npm.taobao.org install
npm run build
```


# yum nodejs 不存在

```
curl --silent --location https://rpm.nodesource.com/setup_16.x | bash -
```

##### 配置npm 仓库
```
npm config set registry https://registry.npm.taobao.org/
```
