## 安装pip
### python 2+ 版本
```
curl "https://bootstrap.pypa.io/pip/2.7/get-pip.py" -o "get-pip.py"
python get-pip.py
```

### python 3+ 版本
```
curl "https://bootstrap.pypa.io/pip/get-pip.py" -o "get-pip.py"
python get-pip.py
```
